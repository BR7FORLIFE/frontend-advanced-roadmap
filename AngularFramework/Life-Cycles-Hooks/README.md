# SignalShop

**Mini e-commerce + Dashboard + Realtime**

## Objetivo

Un solo proyecto que incluye:

- **Catálogo de productos:** Visualiza productos, añade al carrito y realiza el checkout.
- **Dashboard de administración:** Métricas en tarjetas y filtros.
- **Canal en tiempo real:** Comunicación con WebSocket (Signal).
- **Store global:** Implementado con Signals (sin NgRx).
- **Zoneless bootstrapping:** Sin Zone.js.
- **Componentes OnPush:** Uso de `input()` signals.
- **Lifecycle hooks:** Ejemplos concretos de uso.
- **Persistencia:** Guardado en localStorage mediante efectos.
- **Integración externa:** Slider integrado desde `ngAfterViewInit`.
- **Virtual scrolling:** Para listas grandes.
